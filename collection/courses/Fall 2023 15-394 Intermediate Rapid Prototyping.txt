<start course description for 15-394 Intermediate Rapid Prototyping>
Semester: Fall 2023 (aka F23)
Course Description: This course covers additional topics in rapid prototyping beyond the content of 15-294.  Example topics include mechanism design, procedural shape generation using Grasshopper, 3D scanning and mesh manipulation, and advanced SolidWorks concepts. The only prerequisite is basic familiarity with SolidWorks, which can be obtained via 15-294, from other CMU courses, or from online tutorials.
</end course description for 15-394 Intermediate Rapid Prototyping>
