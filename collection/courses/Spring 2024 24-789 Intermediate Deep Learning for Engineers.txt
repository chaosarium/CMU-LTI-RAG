<start course description for 24-789 Intermediate Deep Learning for Engineers>
Semester: Spring 2024 (aka S24)
Course Description: This course will introduce some of the advances in deep learning technology such as sequential learning using Recurrent Neural Networks, Generative Adversarial Networks, Attention models and Transformers, and Diffusion models. Students will learn how to implement these models using deep learning libraries (Pytorch). Introduction to deep learning is the prerequisite.
