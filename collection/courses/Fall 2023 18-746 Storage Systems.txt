<start course description for 18-746 Storage Systems>
Semester: Fall 2023 (aka F23)
Course Description: This course covers the design, implementation, and use of storage systems, from the characteristics and operation of individual storage devices through the OS, database, and networking approaches involved in tying them together and making them useful to key applications' demands and technology trends. Topics to be covered include: network-attached storage, disk arrays, storage networking, storage management, advanced file systems, disk performance enhancement, wide-area data sharing, and storage security. 3 hrs. lec. The class will continue to be like previous years, with the same advanced content and high-level expectations.
</end course description for 18-746 Storage Systems>
