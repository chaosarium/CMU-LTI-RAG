<start course description for 05-839 Interactive Data Science>
Semester: Spring 2024 (aka S24)
Course Description: This course covers techniques and technologies for creating data driven interfaces. You will learn about the entire data pipeline from sensing to cleaning data to different forms of analysis and computation. Technologies used in this course include: Python, Data Science libraries such as Pandas, Visualization tools such as Matplotlib, Seaborn, Altair, Streamlit.
