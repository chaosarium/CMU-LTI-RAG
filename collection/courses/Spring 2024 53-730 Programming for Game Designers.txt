<start course description for 53-730 Programming for Game Designers>
Semester: Spring 2024 (aka S24)
Course Description: Game design sits at the crossroads of many different disciplines--programming, art, writing, design, engineering, psychology, and more. This course takes a practical approach to programming and how it can be used to make one a better game designer. Through individual & collaborative projects, students will learn programming tools, techniques for working with data, methods of working with teams, and prototyping strategies. This course is geared toward non-programmers looking to add programming to their game design toolset.
</end course description for 53-730 Programming for Game Designers>
