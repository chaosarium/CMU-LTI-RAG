<start course description for 05-571 Undergraduate Project in HCI>
Semester: Spring 2024 (aka S24)
Course Description: Experiential learning is a key component of the MHCI program. Through a substantial team project, students apply classroom knowledge in analysis and evaluation, implementation and design, and develop skills working in multidisciplinary teams.  Student teams work with Carnegie Mellon University-based clients or external clients to iteratively design, build and test a software application which people directly use.
