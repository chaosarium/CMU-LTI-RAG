<start course description for 54-939 Entertainment Lighting Programming>
Semester: Fall 2023 (aka F23)
Course Description: Students learn and practice programming techniques on the MA2 series of lighting control consoles. Advanced programming techniques are explored, including media server control and user-defined commands for the console. Different applications are introduced, but the primary focus is on programming for live music performance.
