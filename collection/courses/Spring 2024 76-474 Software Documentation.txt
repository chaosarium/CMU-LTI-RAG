<start course description for 76-474 Software Documentation>
Semester: Spring 2024 (aka S24)
Course Description: This course teaches theory, techniques, and best practices for creating software documentation. We will learn to plan, architect, write, and publish audience-appropriate user assistance, while applying concepts and approaches like minimalism, topic-oriented authoring, single-source publishing, content reuse, and metadata. Students will complete homework assignments and larger projects to reinforce principles and provide experience in all phases of the software documentation lifecycle. Readings and class discussion will bridge theory and practice. This course is for juniors and seniors only.
</end course description for 76-474 Software Documentation>
