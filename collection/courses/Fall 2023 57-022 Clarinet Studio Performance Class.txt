<start course description for 57-022 Clarinet Studio Performance Class>
Semester: Fall 2023 (aka F23)
Course Description: The purpose of this class is to have performance before an audience (studio class members) to ease performance anxiety.  The class meets once a week, is not for credit and is not mandatory.  In addition to playing, the class listens to recordings of various styles of clarinet playing.
