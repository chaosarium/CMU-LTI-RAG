<start course description for 11-634 MCDS Capstone Planning Seminar>
Semester: Spring 2024 (aka S24)
Course Description: This course (open to MCDS students only) is a project based course where students exercise course work in a team based project. The course covers project management, a vision statement, requirements analysis, software architecture, functional specification, and an implementation of a prototype or scientific experiment. In addition, presentation skills are emphasized with multiple presentations throughout the semester.
