<start course description for 46-903 Financial Computing III>
Semester: Spring 2024 (aka S24)
Course Description: In Financial Computing III we implement in C++ the key numerical routines used in the financial industry. The topics include spline interpolation, least-square fitting, finite differences, Fast Fourier Transform, and adjoint differentiation.  As usual, examples and homework exercises will be drawn from finance-related sources.
